J5G3 
====

#### Javascript Graphics Engine ####

J5G3 is an easy to use, extendable and fast canvas-based Javascript game engine.

Building j5g3
-------------

### Rakefile Tasks ###

rake
	Build j5g3 files.
rake minify
	Minifies the code.
rake lint
	Runs jshint on all files.

### Dependencies ###

- jshint
- Ruby/Rake - To build the library.
- UglifyJS (Optional) To minify the code.
